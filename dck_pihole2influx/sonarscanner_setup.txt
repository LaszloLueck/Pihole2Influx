Installation of sonarscanner global:
dotnet tool install --global dotnet-sonarscanner

Send the buildresults to SonarQube on 192.168.1.5
dotnet sonarscanner begin /name:"Pihole2Influx" /d:sonar.host.url="http://192.168.1.5:9000" /k:"3d075ac072aaf2b2b721e086939347b29728c396"
dotnet clean
dotnet build /t:rebuild
dotnet sonarscanner end

To check quality while coding install nuget package
- SonarAnalyzer.CSharp
